志摩国は、現在の三重県鳥羽市の全域と志摩市の大部分（三重県の中東部に位置する半島）に位置する令制国

志摩半島（及び伊勢志摩国立公園）のほぼ中央に位置する「磯部町」は、皇大神宮（内宮）の境外別宮「伊雑宮（いざわのみや）」が鎮座する聖域
* 内宮背後の聖域・伊勢↔︎志摩を繋ぐ山道「島路山（しまじやま）」を越えた神宮の御贄地（神に供える食料を調達する地）
* 伊雑宮は神田を持つ唯一の別宮で、この神田（約1650平方メートル）で行われる御田植祭（6月24日）は国の重要無形文化財・日本三大御田植祭
___

伊勢湾に面する4つの有人離島がある志摩半島の北東端「鳥羽」は、三重県を代表する観光都市の一つ
* 近世以降、志摩国の政治・経済・文化の中心地・鳥羽藩の城下町として発展

熊野灘に面する志摩半島の南部「阿児・志摩」エリアは、真珠養殖発祥の地・伊勢志摩国立公園の中心「英虞湾（あごわん）」に位置する
* 地域を通る唯一の鉄道「近鉄志摩線」の終点「賢島駅」が立地する英虞湾に浮かぶ有人島「賢島」は、2016年に開催された「第42回先進国首脳会議（伊勢志摩サミット）」の開催地であり、「海の軽井沢」と称されるリゾート地
