「皇大神宮（こうたいじんぐう）」は、天照大神の神体「八咫鏡（やたのかがみ）」が奉安される「正宮」と、10の別宮・27の摂社・16の末社・30の所管社のすべてにより成り立つ

内宮（ないくう）と呼ばれる「皇大神宮・正宮」の宮域は、「心のふるさと」と称される日本の原風景が広がる神域
（10の別宮のうち、2宮は正宮の宮域・5宮は正宮の宮域外「月讀宮・倭姫宮」の宮域、3宮は伊勢市外の「瀧原宮・伊雑宮」の宮域にある）

* 日本神話（記紀）において「天照大神（アマテラス）」は、天と地の「天」にあたる神々が住まう「高天原（たかまがはら）」を統べる主宰神
* 高天原の神々の子孫「天孫」が天と地の「地」にあたる「葦原中国（あしはらのなかつくに）」を平定して日本を建国する際に指揮をとった神とされる
（古い原型は伊勢の漁民の磯部氏らが奉じる男性太陽神で、ヤマト王権が皇祖神と同一視して祭るようになったとの見解がある）

「豊受大神宮（とようけだいじんぐう）」は、正宮・4の別宮・16の摂社・8の末社・4の所管社により成り立つ

外宮（げくう）と呼ばれる「豊受大神宮・正宮」の宮域は、天照大神の食事を司る「豊受大御神（トヨウケビメ）」の神域
（4の別宮のうち3宮は正宮の宮域・1宮は正宮の宮域外、摂社・末社・所管社は伊勢市・鳥羽市に分布する）

* 「トヨウケビメ（豊受大神、豊宇気毘売神・等由気太神）」は、古事記に登場するイザナミの尿から生まれた「ワクムスビ」の子と同一とされる
* 元々は丹波国の比治の真奈井（ひじのまない）にいた御饌の神（食物・穀物を司る女神）で、天照大神の神託により外宮の現在地「伊勢国度会の度会郡沼木郷山田原」に呼び寄せられたと伝わる
（鎌倉時代末の伊勢神道では、造化三神の「天之御中主神（あめのみなかぬしのかみ）」と豊受大神を同一視し、これを始源神と位置づけている）

古来より「外宮先祭（げくうせんさい）の礼」といい、両宮を参拝する際は外宮〜内宮の順に参るのがしきたりとされる
* 参拝は朝五時から夕方まで（閉門時間は季節により異なる）
* 神前では「二拝（腰を90度に折り深いお辞儀）二拍手（胸の高さで両手を合わせ、右手先を少し下にずらす）一拝（ずらした指を元に戻し、もう一度深いお辞儀）」で感謝の心をささげる
