「皇大神宮（こうたいじんぐう）」は、天照大神の神体「八咫鏡（やたのかがみ）」が奉安される「正宮」と、10の別宮（8の別宮の所管社）・27の摂社・16の末社・30の所管社の92社すべてにより成り立つ
（10の別宮のうち、2宮は正宮の宮域・5宮は正宮の宮域外「月讀宮・倭姫宮」の宮域、3宮は伊勢市外の「瀧原宮・伊雑宮」の宮域に鎮座）

* 内宮（ないくう）と呼ばれる「皇大神宮・正宮」の宮域は、「心のふるさと」と称される日本の原風景が広がる神域

***

「豊受大神宮（とようけだいじんぐう）」は、正宮・4の別宮・16の摂社・8の末社・4の所管社の計33社により成り立つ
（4の別宮のうち3宮は正宮の宮域・1宮は正宮の宮域外、摂社・末社・所管社は伊勢市・鳥羽市に分布する）

* 外宮（げくう）と呼ばれる「豊受大神宮・正宮」の宮域は、天照大神の食事を司る「豊受大御神（トヨウケビメ）」の神域

古来より「外宮先祭（げくうせんさい）の礼」といい、両宮を参拝する際は外宮〜内宮の順に参るのがしきたりとされる

* 参拝は朝五時から夕方まで（閉門時間は季節により異なる）
* 神前では「二拝（腰を90度に折り深いお辞儀）二拍手（胸の高さで両手を合わせ、右手先を少し下にずらす）一拝（ずらした指を元に戻し、もう一度深いお辞儀）」で感謝の心をささげる
