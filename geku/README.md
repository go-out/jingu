「豊受大神宮（とようけだいじんぐう）」は、正宮・4の別宮・16の摂社・8の末社・4の所管社の計33社により成り立つ

主祭神の「トヨウケビメ（豊受大神、豊宇気毘売神・等由気太神）」は、古事記に登場するイザナミの尿から生まれた「ワクムスビ」の子と同一とされる
元々は丹波国の比治の真奈井（ひじのまない）にいた御饌の神（食物・穀物を司る女神）で、天照大神の神託により外宮の現在地「伊勢国度会の度会郡沼木郷山田原」に呼び寄せられたと伝わる
（鎌倉時代末の伊勢神道では、造化三神の「天之御中主神（あめのみなかぬしのかみ）」と豊受大神を同一視し、これを始源神と位置づけている）

鳥羽市にある1の末社を除く23の摂社・末社は、伊勢市内に分布する

* 摂末社の多くは、明治初期まで伊勢豊受大神宮の神職を世襲していた「度会氏（わたらいうじ）」に関わる神々や、外宮の鎮座以前より度会郡に祀られていた国魂神とされる
