元伊勢のひとつ「甲可日雲宮（こうかひくものみや）」があったとされる滋賀県南東部地域
（読みには「こうか」と「こうが」の2通りがあり、古代に甲賀地域を治めた百済系豪族の鹿深（かふか）氏に由来するとされる）

平安時代には、京都から伊勢神宮の斎宮へ「斎王」の一行が下向する儀式「群行（ぐんこう）」が行わる際に宿泊する仮の宮のひとつ「垂水頓宮」が築かれたほか、745年に聖武天皇によって「紫香楽宮（しがらきのみや）」が造営されるなど、古代に遡る歴史を有する

* 斎王（さいおう）は、天皇が即位するたびに伊勢神宮に奉仕する未婚の皇女または女王
（群行は、近江国の国府と甲賀と垂水・伊勢国の鈴鹿と壱志に設けられた頓宮（とんぐう）で各一泊し、六日目に斎宮へ入る次第だったと伝わるが、実際の行程を語る史料はほとんど残っていない）
