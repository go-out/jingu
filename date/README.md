古来より「豊葦原（とよあしはら）の瑞穂（みずほ）の国」と呼ばれる、水に恵まれ稲が立派に稔る国「日本」を象徴する神宮の祭典は稲作と関わりが深く、田植えから収穫までの周期と共に年間1500回に及ぶ神事が行われる

（毎年定められた日時に行われる祭典は「恒例祭」・天皇皇后両陛下が参拝される際などに行われる祭典は「臨時祭」という）
___

神宮では、古来から続く日本の伝統的な季節の移り変わりに基づく暦「神宮暦」とお神札「神宮大麻」が発行される

* 神宮暦（じんぐうれき）には、明治以後・第二次世界大戦以前に日本の正暦（公式な歴）と定められていた「本暦・大暦（たいれき）」と、一般向けに簡略化された「略本暦・小暦（しょうれき）」の二種がある
* 神宮大麻（じんぐうたいま）は、天照大神を祀るお神札で、毎年十月から十二月に全国の神社から各家庭に頒布される

***

一月上旬、神宮大麻（神宮の神札）と神宮暦の奉製始めの儀式「大麻暦奉製始祭」
（この祭典を初めに、発行された歴が神社本庁に属する全国の神社に配られ、三月五日に「大麻暦頒布終了祭」が行われる）

四月中旬、神宮大麻の用材伐り始めの儀式「大麻用材伐始祭」
（宇治橋に近い祭場にて神宮大麻の奉製を祈願する祭儀が行われ、この祭典を初めに「神宮大麻」が随時奉製、十二月下旬に「大麻暦奉製終了祭」が行われる）

九月十七日、神宮大麻の頒布始めの儀式「大麻暦頒布始祭」
（この祭典を初めに、全国に神社庁や神社に「神宮大麻」が配られ、全国各地の神社から家庭へ頒布される）
