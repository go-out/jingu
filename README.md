伊勢神宮は、三重県伊勢市にある「内宮（ないくう）・外宮（げくう）」の通称で知られる2つの正宮（しょうぐう）「二所大神宮（皇大神宮・豊受大神宮）」と、14の別宮・109の摂社・末社・所管社の合計125社から成り立つ集合体、日本最高位・特別格の宮
（正式名称は「神宮（じんぐう）」だが、他の神宮と区別するため、一般には「伊勢」の地名を冠する）

* 別宮（べつぐう）は、正宮に次いで尊いとされる「正宮のわけみや」（瀧原宮・伊雑宮の宮域にある3宮を除く、11の別宮は伊勢市内に分布）
* 43の摂社（せっしゃ）・24の末社（まっしゃ）・42の所管社（しょかんしゃ）は、三重県内の4市2郡（伊勢市・松阪市・志摩市・鳥羽市、度会郡の大紀町・玉城町・度会町、多気郡多気町）に分布する

神宮の創祀は3世紀後半から4世紀前半、天照大神の神霊を込めた三種の神器の一つ「八咫鏡（やたのかがみ）」を鎮座する珠玉の地を求めて、崇神天皇（第10代、すじん）の皇女「豊鍬入姫命（とよすきいりひめのみこと）」〜 垂仁天皇（第11代、すいにん）の第四皇女「倭姫命（やまとひめのみこと）」が日本各地を巡った末、ついにたどり着いた現在地「神風吹く伊勢の五十鈴川のほとり（五十鈴宮）」が永遠の社に定められた

* 豊鍬入姫命が54年・倭姫命が34年、延べ88年の長旅の中で一時的に天照大神が祀られた場所は「元伊勢（もといせ）」と呼ばれ、関西周辺に元伊勢の伝承を持つ場所が多数存在する（正確な場所はほとんど不明確）
